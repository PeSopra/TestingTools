INTRODUCTION
------------
This module help us to integrate AI R&R Moderation.

INSTALLATION
------------
Installation of this module is just like any other Drupal module. Visit
https://www.drupal.org/node/1897420 for further information.

- Go to drupal extend and enable the module

CONFIGURATION
-------------
1. Go to configuration ln_nesgen `admin/config/lightnest/nesgen`
2. Go to `admin/config/lightnest/nesgen/rr`

REQUIREMENTS
-------------
- dsu_ratings_reviews
- ln_nesgen

FUNCTIONALITY
-------------

- This module integrates AI R&R Moderation and provides this extra data to the server.
It can serve as an ideal complement to Engage RR. This module allows you to respond
to comments automatically thanks to its integration with NesGen AI

HOW TO USE
-------------
- Enable and configure to start working.
When a user makes a comment on R&R if the result is satisfactory, this module will
give an automatic response generated by NesGen AI
